# postnummer
## Template for getting Norwegian postal names from postal codes.

###### Requires:
> jQuery

###### Simple Syntax

```html
<input id="postnr">
<input id="poststed">
```
###### Add jQuery and postnummer.js
```html
<script src="https://code.jquery.com/jquery-3.1.1.min.js"></script>
<script src="js/postnummer.js"></script>
```
